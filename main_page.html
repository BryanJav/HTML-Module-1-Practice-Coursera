HEADING ELEMENTS

< header > show the header
< section > creates a separate section inside the page
< article > article tag, used mainly for articles and comments

< div > division tag, this is mainly used for dividing areas into a section for ease of readability and also for CSS editing, later on,
< div > div tags are block elements, meaning they end on a new line

< span > span tag, this is used when you want to highlight different elements within a tag without altering its position or creating a new line at the end
< span > span tags are inline elements, meaning they dont end in a new line

Header tags: Arrange by most important tag to the least important tag
  1. < h1 >
  2. < h2 >
  3. < h3 >
  4. < h4 >
  5. < h5 >
  6. < h6 >

< ul > unordered lists, makes a list of objects
< li > use this to put bullets on the objects of the lists
< ol > ordered lists of objects, output lists are 1. 2. 3. And so on
< li > use this to put the numbers on the objects of the lists
< p > paragraph tags, used for paragraphs with a article

CHARACTER ENTITY ELEMENTS

3 Characters You Should Avoid!!
“<” = instead use “< ;”, always include a semicolon in the end
“>” = instead use “> ;”
“&” = instead use “& ;”

THIS APPLIES EVEN TO TEXT

& copy ; = Copywrite logo
& nbsp ; = non-breaking space (if you want the text to always stay and not break the line)
& quote ; = takes a quote in the text and makes it compatible on other versions of meta

LINKS

< a > < /a >
Same directory < a > :
<a href = “same-directory.html” title=”same dir link” > (this is the hyperlink text) linking to a file in the same directory </a>

Website link <a>:
<a href= “website link” (this is to create a new tab) -> target=”_blank” (this is the title when the mouse hovers on the link) -> title=”Like Our Page”> Coursera Facebok Page </a>

Section link <a> :
id= ”name of a section of the page”

“ # “ = SECTION IDENTIFIER

USED FOR SPA (SINGLE PAGE APPLICATIONS) ALWAYS USE A # TO DETERMINE THE PAGE ID
EXAMPLE: id=”section1” or name=”section6”

THEN INCLUDE A " # " INSIDE THE HREF TO DETERMINE THE ID OR NAME
<li> <a href=”#section1”> #section1 </a> </li>

DISPLAYING IMAGES

ALWAYS SPECIFY THE WIDTH AND HEIGHT OF THE IMAGE IN ORDER TO NOT BREAK THE LAYOUT OF THE WEBSITE

<img src> is an inline element

<img src=” image external link or image source ” width=” width of the image ” height=” height of the image ” alt=”screen reader text (this is optional but it is good practice to do so)”>
